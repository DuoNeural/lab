# DuoNeural Lab

Open research lab. Post-training, abliteration, GRPO, quantization on open-weights models.

**Active experiments:**
- `experiments/ctm_baseline_comparison.py` — CTM+GHL v5 vs baseline transformer A/B test (running)
- `experiments/idvro_dataset_gen.py` — ID-VRO ATC synthetic dataset generator (Gemma-4-31B via OpenRouter)

**Tools:**
- `tools/lab_watchdog.sh` — SSH-based lab monitor w/ Telegram notification on completion
